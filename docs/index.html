WealthyWidgets

12 accessible, customisable React components for rapid UI development.

npm install wealthywidgets

Getting Started

Import components and styles, then use them anywhere in your React app.

Installation
# 1. Install npm install wealthywidgets # 2. Import styles once in your app root import 'wealthywidgets/dist/styles.css'; # 3. Use components import { Button, Card, Modal } from 'wealthywidgets';
Toast setup (one-time)
import { ToastProvider } from 'wealthywidgets'; function App() { return ( <ToastProvider position="top-right"> <YourApp /> </ToastProvider> ); }
